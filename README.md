## 📌 Project Overview
A self-contained Python application that uses the **A* Search Algorithm** to find the absolute shortest path through a 2D grid maze.The project handles complex grid obstacles, supports custom heuristics, and visualizes pathfinding operations both in the console and via an interactive desktop user interface.

## 🚀 Key Features
**Intelligent Pathfinding:** Custom A* implementation utilizing Manhattan and Euclidean heuristic distance calculations.
* **Robust Logic:** Gracefully manages edge cases including blocked paths, empty grids, and unreachable targets.
* **Dual Visualization:** Supports clean, text-based console layouts alongside fully animated maze solving windows using the `pyamaze` library.

## 📂 Repository Package Contents
* **`Syntecxhub_Maze_Solver_A_Star.zip`**: A manual deployment archive containing:
